# Innovation and Entrepreneurship Project

Coursework and deliverables for the PGI activity in the Innovation and
Entrepreneurship course (Universidade de Coimbra). The centerpiece is the
final group project report, a proof-of-concept innovation activity covering
value proposition, market, business model, and intellectual property.

## Repository structure

```
.
├── report/         Final project report (the main deliverable)
├── templates/      Submission templates used to build the report
├── lectures/       Course lecture material
└── exam-prep/      Study material for the written exam
```

### report/
- `final-report.pdf` is the final submitted group report. This is the main
  deliverable of the project.

### templates/
- `final-report-template.docx` is the structure provided for the final report
  (value proposition, market, business model, and so on).
- `proof-of-concept-template.docx` is the template for the proof-of-concept and
  MVP section.

### lectures/
- `intellectual-property.pdf` is the guest lecture on intellectual property and
  the protection of intellectual assets, given by Instituto Pedro Nunes (IPN).

### exam-prep/
- `2018-normal-exam.pdf` is a past exam used for practice.
- `summaries/` holds one-page topic summaries covering management and
  innovation, business models, startups, design thinking, IP protection, and
  open innovation. Files are numbered by course topic.

## Notes

The report is image-based (exported via Print to PDF), so its text is not
selectable. Open it in any PDF viewer to read it.
